# Прокси для обхода блокировки API (за 3 минуты)

Если приложение не может достучаться до нейросетей (так бывает в РФ и ещё ряде стран), поднимите бесплатный прокси на Cloudflare. Это просто — нужно один раз зарегистрироваться и вставить готовый код.

> Что это даёт: приложение начнёт ходить к Gemini и NanoGPT через ваш личный сервер-прокси, который нигде не заблокирован. Бесплатно в рамках лимитов Cloudflare (100 000 запросов в день — для личного использования более чем достаточно).

## Шаг 1. Зарегистрируйтесь на Cloudflare

1. Откройте https://dash.cloudflare.com/sign-up
2. Введите email и придумайте пароль.
3. Подтвердите email по ссылке из письма.

> Регистрация бесплатная, банковская карта **не нужна**.

## Шаг 2. Создайте Worker и вставьте код

1. В панели Cloudflare слева откройте **Workers & Pages**.
2. Нажмите **Create** → **Create Worker**.
3. Придумайте имя (например, `nutrilens-proxy`) → нажмите **Deploy**.
4. Нажмите **Edit code** (редактировать код).
5. **Удалите** весь текст, который там есть, и **вставьте** содержимое файла `worker.js` из этой же папки.
6. Нажмите кнопку **Deploy** (справа сверху).

## Шаг 3. Скопируйте адрес прокси

После деплоя наверху страницы будет адрес вида:

```
https://nutrilens-proxy.ваш-логин.workers.dev
```

Запомните его — это ваш адрес прокси.

## Шаг 4. Вставьте адрес в приложение

В приложении откройте **Настройки → Прокси / обход блокировок** и заполните два поля:

| Поле                     | Значение                                  |
|--------------------------|-------------------------------------------|
| URL прокси для Gemini    | `https://nutrilens-proxy...workers.dev/gemini` |
| URL прокси для NanoGPT   | `https://nutrilens-proxy...workers.dev/nano`   |

То есть к вашему адресу добавьте в конце `/gemini` и `/nano` соответственно.

Нажмите **Сохранить**. Готово — можно пользоваться.

---

## Частые вопросы

**Обязательно ли это?**
Нет. Если приложение и так работает — поле оставьте пустым, прокси не нужен.

**Безопасно ли это? Мои ключи нигде не сохранятся?**
Да, безопасно. Прокси просто перенаправляет запросы и ответы, **не хранит** и не логирует ваши API-ключи — они идут напрямую от приложения к нейросети.

**Как проверить, что прокси работает?**
Откройте в браузере `https://nutrilens-proxy...workers.dev/gemini/v1beta/models`. Если в ответ пришёл JSON (пусть даже с ошибкой) — прокси работает. Если страница не грузится — проверьте, что вы вставили весь код и нажали Deploy.

**Хочу, чтобы моим прокси пользовался только я.**
В Cloudflare откройте воркер → **Settings** → **Variables and Secrets** → добавьте переменную `PROXY_TOKEN` с любым длинным паролем. Это дополнительная защита, но в самом приложении её поддержка пока не сделана — токен придётся отправлять вручную. Для личного использования можно оставить прокси открытым.
